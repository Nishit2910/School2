# Bright Future School Website

A modern, responsive school information website built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Interactive Navigation**: Smooth scrolling and active link highlighting
- **Animated Elements**: Scroll animations and transitions for better user experience
- **Sections Include**:
  - Home/Hero section with call-to-action
  - About section with school statistics
  - Teachers showcase
  - Admission process details
  - Fee structure with pricing cards
  - Contact information and form

## Technologies Used

- HTML5
- CSS3 (with animations and gradients)
- JavaScript (ES6+)
- No external frameworks required

## Deployment

This website is deployed on GitHub Pages.

## Local Development

1. Clone the repository
2. Open `index.html` in your web browser
3. No build process required!
